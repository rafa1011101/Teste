# 🐓 Frangorilla — Aventura no Brasil

Jogo de plataforma estilo Mario, num único arquivo HTML, sem dependências
externas. Roda no navegador do computador e no iPhone.

## Como jogar

Abra o arquivo `index.html` no navegador. Não precisa instalar nada.

### Controles

- **Computador:** setas ou `A`/`D` para andar, `Espaço` ou `W` para pular
  (segure para pular mais alto), `Enter` para começar, `M` liga/desliga a música.
- **iPhone / celular:** toque na tela para começar e use os botões de toque
  (setas à esquerda, "PULO" à direita). De preferência com o aparelho na horizontal.

## O jogo

- Personagem: o galo **Frangorilla** com a camisa da seleção brasileira.
- Cenário do Brasil: Corcovado com o Cristo Redentor, floresta e castelo.
- Fase Mundo 1-1 com goombas, tartarugas, plantas carnívoras, moedas,
  blocos "?", canos, buracos com água e a bandeira do Brasil no final.
- Trilha sonora e efeitos sonoros gerados em tempo real (WebAudio), sem arquivos.

A cada 50 moedas você ganha uma vida extra. O objetivo é chegar à bandeira
perto do castelo.
